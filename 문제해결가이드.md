# 🚨 웹사이트가 열리지 않을 때 해결 방법

## 1단계: 기본 확인

### ✅ 파일 위치 확인
프로젝트 폴더에 다음 파일들이 있는지 확인하세요:
- `app.py`
- `crowd_detect.py`
- `density.py`
- `direction.py`
- `utils.py`
- `requirements.txt`

### ✅ 패키지 설치 확인
명령 프롬프트(cmd) 또는 PowerShell에서:
```bash
pip list | findstr streamlit
```
Streamlit이 보이지 않으면:
```bash
pip install streamlit ultralytics opencv-python numpy pandas Pillow
```

## 2단계: 실행 방법

### 방법 A: 배치 파일 사용 (추천)
1. `START_HERE.bat` 파일을 **더블클릭**
2. 자동으로 브라우저가 열립니다

### 방법 B: 수동 실행
1. 프로젝트 폴더에서 **Shift + 우클릭** → "여기서 PowerShell 창 열기"
2. 다음 명령어 입력:
   ```bash
   streamlit run app.py
   ```

## 3단계: 오류별 해결 방법

### ❌ 오류: "streamlit을 찾을 수 없습니다"
**해결:**
```bash
pip install streamlit
```

### ❌ 오류: "모듈을 찾을 수 없습니다" (ModuleNotFoundError)
**해결:**
```bash
pip install -r requirements.txt
```
또는 개별 설치:
```bash
pip install ultralytics opencv-python numpy pandas Pillow
```

### ❌ 오류: "포트가 이미 사용 중입니다" (Port already in use)
**해결:**
1. 다른 터미널/명령 프롬프트 창을 모두 닫으세요
2. 또는 다른 포트 사용:
   ```bash
   streamlit run app.py --server.port 8502
   ```
3. 브라우저에서 `http://localhost:8502` 접속

### ❌ 오류: "파일을 찾을 수 없습니다" (File not found)
**해결:**
1. `app.py` 파일이 있는 폴더로 이동했는지 확인
2. 한글 경로 문제일 수 있음 → 폴더 이름을 영문으로 변경 시도

### ❌ 브라우저가 자동으로 열리지 않음
**해결:**
1. 터미널에 표시된 URL 확인 (예: `http://localhost:8501`)
2. 브라우저에서 해당 URL을 직접 입력
3. 또는 `http://127.0.0.1:8501` 시도

### ❌ "YOLO 모델 로딩 중..." 에서 멈춤
**해결:**
- 첫 실행 시 YOLO 모델 다운로드 중 (약 6MB)
- 인터넷 연결 확인
- 몇 분 기다려보세요

## 4단계: 수동 테스트

### Python이 제대로 작동하는지 확인:
```bash
python --version
```

### Streamlit이 설치되었는지 확인:
```bash
python -c "import streamlit; print('OK')"
```

### 모듈 import 테스트:
```bash
python test_run.py
```

## 5단계: 완전 재설치 (최후의 수단)

```bash
# 1. 모든 패키지 재설치
pip install --upgrade streamlit ultralytics opencv-python numpy pandas Pillow

# 2. 캐시 정리
pip cache purge

# 3. 다시 실행
streamlit run app.py
```
